# Eduardo Succo 🌟
### **Engenheiro de Dados Sênior**

Sou **Engenheiro de Dados Sênior** com foco em automação de processos, construção de data pipelines e entrega de dados confiáveis para consumo analítico. Tenho sólida experiência em bancos de dados e SQL, atuando com modelagem, otimização de desempenho e integração de dados em ambientes corporativos.

Atuo no desenvolvimento e sustentação de processos **ETL/ELT** com Apache Airflow, Python e Spark/PySpark em ambiente Hadoop, com atenção rigorosa à escalabilidade e operação em produção. Tenho experiência na definição de arquiteturas de dados alinhadas a requisitos de escala, qualidade e governança, aplicando boas práticas de monitoramento para manter a estabilidade operacional.
